# New-server
Cервер на Node.js с использованием модулей Http и Fs, запускается по адресу 127.0.0.1:3003.

## Написан обработчик входящего запроса:
Ответом на запрос ?hello=<name> является строка "Hello, <name>", код ответа 200.
Если параметр hello указан, но не передано <name>, то ответом является строка "Enter a name", код ответа 400.
Ответом на запрос ?users является JSON с содержимым файла data/users.json, код ответа 200.
Если никакие параметры не переданы, то ответом является строка "Hello, World!", код ответа 200.
Если переданы какие-либо другие параметры, то будет пустой ответ, код ответа 500.
